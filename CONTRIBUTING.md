# Contributions

Career OS is published as a read-only template. The canonical repository does not accept pull requests, user files, code changes, documentation changes, or other contributions.

To use it, copy the template and make changes only in your own private copy. Do not send personal career data to the canonical repository.

This boundary keeps the public template stable and prevents personal instances from being mixed with the source.

## Feedback is welcome

You can still help improve Career OS by [commenting in the feedback
thread](https://github.com/allanc92/Career-OS/discussions/1) or [starting a new
idea](https://github.com/allanc92/Career-OS/discussions/new?category=ideas).
Useful feedback includes:

- what was easy or difficult during setup
- where onboarding felt clear or confusing
- which workflows helped during a real job search
- missing use cases or instructions
- ideas that would make Career OS safer or more useful

Do not include resumes, application materials, employer-confidential
information, or other personal career data. Describe the workflow or problem
without sharing sensitive details.
