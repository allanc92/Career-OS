# Jordan Rivera example

Jordan Rivera and every organization, role, date, metric, and event in this
folder are fictional. They exist only to show how Career OS files connect.
Never treat this folder as evidence about a real user.

This example shows:

- a completed career profile
- a grounded master resume
- one achievement linked to a StoryBank entry
- a StoryBank pipeline with another developing story
- one opportunity moving from research to application preparation
- application notes and a tailored resume grounded in the fictional sources

Compare these files with the blank schemas in the repository root. A real
Career OS stores the corresponding files outside `examples/`, where
`.gitignore` keeps them private by default.
